# AI-FaceAttendance
 📸 Face Recognition Attendance System  

🚀 **Automated Attendance Tracking using Python & OpenCV**  
A practical project that integrates **Computer Vision** and **AI** to recognize faces in real‑time and mark attendance securely & efficiently.  

## 🛠️ Tech Stack
- 🐍 **Python** – Core programming language  
- 👁️ **OpenCV** – Face detection & recognition  
- 🗄️ **SQLite/MySQL** – Attendance database  
- 🌐 **Flask** – Web interface & templates  

## 📂 Project Structure
```
Face-Recognition-attendance-system/
│── static/
│   ├── images/        # 📸 Store student images
│── templates/
│   ├── index.html     # 🌐 Frontend UI
│── app.py             # ⚙️ Main application logic
│── database.py        # 🗄️ Database operations
│── requirements.txt   # 📦 Dependencies
```
## ⚙️ Features
- 🔐 **Face Registration** – Store student images for recognition  
- 📡 **Real‑Time Detection** – Capture faces via webcam  
- 📝 **Attendance Logging** – Save records automatically in database  
- 📊 **Web Dashboard** – Simple UI for attendance visualization  

## 🚀 Getting Started
1. 📥 Clone the repo:  
   ```bash
   git clone https://github.com/yourusername/Face-Recognition-attendance-system.git
   ```
2. 📦 Install dependencies:  
   ```bash
   pip install -r requirements.txt
   ```
3. ▶️ Run the app:  
   ```bash
   python app.py
   ```
4. 🌐 Open browser at `http://127.0.0.1:5000/`
